# TestGame

このリポジトリは `TestGame` プロジェクトのソースコードおよびアセットを管理しています．

## 🎮 ゲーム概要 (Game Overview)

### 基本情報 (Basic Info)
- **タイトル (仮)**: うらおもて
- **ジャンル**: 2D 2人プレイ専用 協力アクションゲーム
- **プラットフォーム**: PC
- **想定プレイ時間**: 1ステージ 30秒〜10分

### 企画の核 (Core Concept)
- **一言コンセプト**: 「2人の息を合わせる」ことがすべての鍵となる、完全協力特化型アクション．
- **ターゲット層**: 友人や恋人、配信者など、誰かと一緒にワイワイ盛り上がりながら遊びたいプレイヤー．
- **プレイヤー体験**: 失敗を重ねながら「せーのっ！」と声を掛け合い、ギリギリでギミックを突破した時の「圧倒的な一体感と達成感」を2人で共有する体験．
- **リファレンス（参考ゲーム）**: PICOPARK, Ambidextro

### メカニクス (Game Mechanics)
- **プレイヤーループ**: ステージのギミックを観察・相談する ➡️ タイミングを合わせて同時にアクションを起こす ➡️ 協力して道を切り開きゴールを目指す
- **特徴的なシステム (USP)**:
  - **焦りがミスを誘う「電撃バー」**: 常に背後から時間経過で電撃バーが迫ってくるため，擬似的な強制スクロールのような状態になる．「早く！」という焦燥感がミスコミュニケーションを引き起こす．
  - **同時入力で「左右反転」**: このゲームの最大の鬼門．2人が**同時に**移動操作を行うと，左右の移動方向が反転する（例: 2人とも左に逃げようとして左を押すと，どちらも右に進んでしまう）．そのため「ここは俺だけが動く！」「せーので右！」といった高度な声かけが必須となる．

### ビジュアル・サウンド (Aesthetics)
- **アートスタイル**: ギミックやキャラクターの視認性を高めるため、装飾を抑えた洗練された「シンプル」なカラーパレットとビジュアルを採用．
- **サウンド方針**: コミュニケーション（声かけ）の邪魔にならない、プレイに集中できるサウンドデザイン．
