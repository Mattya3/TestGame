# TestGame チーム開発ドキュメント

このディレクトリには、チームメンバーがスムーズに共同開発を進めるためのガイドラインやルールがまとめられています。
開発を本格的に始める前に、各自目を通してください。(面倒かと思いますが、チーム開発の練習だと思って読んでみてください。)
> これらのドキュメントは生成AIを利用して構成・作成されました。そのため、開発の状況に応じてルールが変更される可能性は十分にあります。
